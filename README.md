# ai-helper

**AI-Powered Personal Automation Tool & Self-Improving Next.js Template**

A modern full-stack foundation with autonomous AI capabilities.

## Quick Start

```bash
pnpm install
cp .env.example .env
pnpm dev

Features

Self-evolving AI agent using Groq and custom orchestration
Drizzle ORM + PostgreSQL ready
Next.js 16 (App Router) + React 19 + TypeScript 5
Tailwind CSS 4 + full shadcn/ui components
Playwright E2E testing
Docker, docker-compose, and systemd support
GitHub Actions (CI, Security, Self-Improvement)

Useful Commands

pnpm db:generate — Generate migrations
pnpm db:migrate — Run migrations
pnpm test — Run Playwright tests
pnpm build — Production build

See docs/AI_GUIDE.md for AI development conventions.
